## 🔍 About the Projects

This repository tracks my journey learning and mastering Power BI—from my first hands-on experience to building advanced, domain-specific business intelligence dashboards.

### Dashboards Overview

#### 1. Hospital Management Dashboard (`Hospital_Management_Dashboard.pbix`)
* **Objective:** To monitor healthcare operations, patient admissions, resource allocation, and key performance indicators (KPIs) for hospital administration.
* **Key Insights:** Tracks patient stay durations, department-wise load, admission trends, and staff/bed utilization metrics.

#### 2. Indian Premier League (IPL) Analysis (`Indian Premier League (IPL).pbix`)
* **Objective:** A comprehensive sports analytics dashboard tracking IPL cricket tournament data across multiple seasons.
* **Key Insights:** Features batsman and bowler performance metrics, team standings, win/loss trends, toss impact, and venue-based statistics.

#### 3. Product Dashboard (`product_dashboard.pbix`)
* **Objective:** Designed for retail/e-commerce product sales analysis.
* **Key Insights:** Monitors product-wise sales revenue, profit margins, inventory levels, return rates, and top-performing product categories.

#### 4. Practice & Learning Milestones
* **`DAX_practice_task.pbix`:** A dedicated file containing advanced DAX calculations, calculated columns, measures, time intelligence functions, and filter contexts.
* **`First Day on PowerBI.pbix` & `Day 2.pbix`:** Baseline projects highlighting initial learning milestones, fundamental drag-and-drop visuals, and data loading.
* **`Mini Project.pbix` & `Dashboard.pbix`:** Structured intermediate milestone projects focusing on data cleaning, basic star-schema data modeling, and UI/UX design fundamentals.

---

## 🛠️ Key Skills Demonstrated

* **Data Transformation (Power Query):** Cleaning, shaping, merging/appending queries, and handling null values.
* **Data Modeling:** Creating robust relationships (1-to-many), creating Star Schemas, and optimizing data models.
* **DAX (Data Analysis Expressions):** Writing calculated measures and columns using functions like `CALCULATE`, `SUMX`, `DIVIDE`, and Time Intelligence (`SAMEPERIODLASTYEAR`, `YTD`, etc.).
* **Data Visualization & UI/UX:** Utilizing custom visuals, bookmarks, page navigation buttons, drill-throughs, and cohesive color themes for user-friendly interactivity.
